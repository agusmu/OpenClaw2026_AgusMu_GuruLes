# GuruLes 🤖📚

> Autonomous AI Learning Companion for Junior High School Students

GuruLes adalah AI pendamping belajar berbasis Telegram yang dirancang untuk membantu siswa SMP belajar secara lebih teratur, personal, dan mandiri. Berbeda dengan chatbot belajar biasa, GuruLes mampu menjalankan workflow belajar secara proaktif seperti mengingatkan jadwal sekolah, mengecek PR, menawarkan sesi belajar, dan menjaga konteks interaksi melalui persistent memory.

Project ini dibuat dari kebutuhan nyata seorang ayah yang ingin tetap mendampingi proses belajar anaknya meskipun harus kembali fokus bekerja.

---

# ✨ Features

## 🕐 Autonomous Daily Learning Workflow

GuruLes dapat menjalankan rutinitas belajar secara otomatis:

- Reminder bangun pagi
- Reminder jadwal & buku sekolah
- Reminder PR sepulang sekolah
- Reminder belajar malam

---

## 📚 Multiple Learning Modes

### 📖 Materi Sederhana

Penjelasan step-by-step dengan bahasa santai dan mudah dipahami.

### 🎴 Flashcard

Evaluasi cepat menggunakan pertanyaan benar/salah.

### 📝 Latihan Soal

Soal pilihan ganda dengan simulasi waktu pengerjaan.

### 🙋‍♀️ Belajar Bareng

Siswa bebas bertanya seperti sedang belajar dengan guru les pribadi.

---

## 🧠 Persistent Memory

GuruLes menggunakan file-based memory untuk mengingat:

- jadwal sekolah
- preferensi belajar
- reminder
- konteks percakapan
- workflow belajar pengguna

---

## 👨‍👩‍👧 Telegram Personal & Family Group Support

GuruLes dapat digunakan melalui:

- chat personal Telegram
- group keluarga Telegram

Sehingga orang tua dan anak dapat memantau proses belajar bersama-sama.
